## Цель работы
Разработать прототип модуля для анализа и обработки сетевых пакетов с возможностью фильтрации по заданным критериям. Студенты получат практический опыт работы с сетевыми протоколами и структурами данных для обработки сетевого трафика.

## Постановка задачи

2. Анализирует заголовки пакетов (Ethernet, IP, TCP/UDP)
3. Применяет заданные фильтры
4. Формирует статистику по отфильтрованному трафику

## Контракт ввода-вывода

### Входные данные:
- Имя сетевого интерфейса
- Конфигурационный файл с правилами фильтрации

- Лог-файл с информацией о перехваченных пакетах

  - Общее количество обработанных пакетов
  - Количество пакетов по протоколам (TCP/UDP/ICMP/другие)
  - Топ-5 IP-адресов отправителей
  - Топ-5 IP-адресов получателей
  - Средний размер пакета

1. Использование библиотеки libpcap (для C/C++) или Scapy (для Python)
2. Поддержка фильтрации по следующим полям:
   - IP-адрес источника/назначения
   - Порт источника/назначения
   - Протокол транспортного уровня
   - Размер пакета (больше/меньше заданного значения)

3. Обработка следующих исключительных ситуаций:
   - Неверный формат конфигурационного файла
   - Отсутствие прав доступа к сетевому интерфейсу
   - Превышение лимита обрабатываемых пакетов (10000 пакетов/сек)

### Вариантные параметры (уникальные для каждого студента):
1. Дополнительные поля для фильтрации:
   - Флаги TCP (SYN, ACK, RST и т.д.)
   - TTL пакета
   - Тип ICMP-сообщения

2. Дополнительная статистика:
   - Распределение пакетов по времени суток
   - Обнаружение аномальной активности
   - Геолокация IP-адресов

3. Дополнительные функции:
   - Сохранение дампа пакетов в формате pcap
   - Визуализация статистики в реальном времени
   - Уведомления о подозрительной активности

## Ограничения
- Время работы программы: не более 5 минут
- Максимальный размер лог-файла: 10 МБ
- Поддержка только IPv4
- Минимальная версия Python: 3.8 или C++17

1. Корректность работы (40%)
2. Полнота реализации требований (30%)
3. Обработка ошибок (15%)
4. Качество кода и документация (15%)

1. Stevens W.R. TCP/IP Illustrated, Volume 1
2. Официальная документация libpcap/Scapy
3. RFC 791 (IP), RFC 793 (TCP), RFC 768 (UDP)
