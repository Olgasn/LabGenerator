- Поддержка только текстовых файлов (расширения .txt, .csv, .log)

### Граничные случаи:
- Попытка отправить файл, который уже существует на сервере
- Прерывание передачи файла
- Передача пустого файла
- Длинные имена файлов (>255 символов)
