- Возвращает ответ в формате: `СТАТУС:ДАННЫЕ`
- Поддерживает множественные одновременные подключения

3. Технические требования:
   - Язык программирования: Python 3.8+
   - Использование стандартной библиотеки (без внешних зависимостей)
   - Код должен быть документирован (комментарии и docstrings)
   - Обработка исключений для сетевых операций

### Ограничения:
- Максимальный размер одного сообщения: 1024 байта
- Таймаут соединения: 30 секунд
- Поддержка только текстовых данных в кодировке UTF-8

### Граничные случаи:
- Потеря соединения во время передачи данных
- Некорректный формат команды
