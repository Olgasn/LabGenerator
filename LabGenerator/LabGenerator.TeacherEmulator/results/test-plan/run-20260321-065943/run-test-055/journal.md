   - Класс `Order` с полями: id, список товаров, статус, дата создания
   - Класс `Warehouse` для управления состоянием склада

   - Добавление/удаление/изменение товаров

   - Изменение статуса заказов

     - Список товаров с низким запасом
     - Отчет по продажам за период
     - Остатки по категориям

3. **Обработка ошибок**:
   - Проверка корректности ввода
   - Обработка недостатка товара при заказе
   - Восстановление после сбоев

### Варианты индивидуальных заданий:
1. **Тип склада**:
   - Обычный склад
   - Холодильный склад (учет температуры хранения)
   - Опасные материалы (учет класса опасности)

2. **Дополнительная функциональность**:
   - Система скидок
   - Резервирование товаров
   - История изменений цен
   - Мультиязычный интерфейс

3. **Способ хранения данных**:
   - Текстовые файлы
   - База данных SQLite
   - Сериализация объектов

### Ограничения:
- Использовать только стандартную библиотеку Java (или выбранного языка)
- Минимум 3 уровня наследования
- Применение полиморфизма и инкапсуляции
