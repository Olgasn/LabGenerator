Разработать объектно-ориентированную систему для моделирования и анализа экспериментов по изучению физических свойств различных материалов при различных условиях.

## Постановка задачи
Создать приложение для исследования зависимости физических характеристик материалов от внешних условий, включая:
1. Моделирование эксперимента с настраиваемыми параметрами материала и условий
2. Генерацию экспериментальных данных с учетом погрешностей измерений
3. Статистическую обработку результатов
4. Визуализацию зависимостей

## Контракт ввода-вывода

### Входные данные:
- Параметры материала (плотность, удельная теплоемкость, теплопроводность)
- Внешние условия (температура, давление, влажность)
- Количество измерений (от 100 до 1000)
- Тип исследуемой зависимости (тригонометрическая, логистическая, полиномиальная)

### Выходные данные:
1. Таблица сгенерированных данных измерений
2. Графики зависимостей физических характеристик от условий
3. Статистический отчет (мода, стандартное отклонение, коэффициент детерминации)
4. Аналитический вывод о значимости результатов

## Требования к реализации

### Обязательные классы:
1. `MaterialExperiment` - основной класс эксперимента
2. `MaterialProperty` - класс свойств материала
3. `ConditionGenerator` - генератор условий эксперимента
4. `StatisticalAnalyzer` - класс статистического анализа
5. `PropertyVisualizer` - визуализатор результатов

### Функциональность:
- Генерация данных с нормальным распределением погрешностей (8-12%)
- Реализация трех типов зависимостей: тригонометрической, логистической, полиномиальной
- Расчет статистических показателей: мода, стандартное отклонение, R²
- Построение 3D-графиков и тепловых карт зависимостей
- Экспорт результатов в текстовый файл

### Ограничения:
- Использование только стандартной библиотеки Python (кроме matplotlib для визуализации)
- Минимум 100 испытаний
- Погрешность измерений: 8-12%

### Граничные случаи:
- Критические точки фазовых переходов
- Предельные значения температур и давлений
- Нулевые значения параметров
- Корректная обработка выбросов в данных
