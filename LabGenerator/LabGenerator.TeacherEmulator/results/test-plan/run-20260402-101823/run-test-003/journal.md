- Устанавливает соединение с сервером по указанному адресу и порту

- Выводит результаты взаимодействия

4. Обработка ошибок сетевого взаимодействия

### Функциональные требования:
- Реализовать простой текстовый протокол прикладного уровня
- Поддержка как минимум 3 типов запросов (например, эхо-запрос, запрос времени, запрос статистики)
- Валидация входящих данных
- Корректное завершение работы при получении сигналов ОС

### Ограничения:
- Запрещено использовать высокоуровневые фреймворки (только базовые сетевые библиотеки)
- Приложение должно работать в гетерогенной сети
- Код должен быть документирован в соответствии с принятыми стандартами

- Некорректные или поврежденные пакеты
- Попытка подключения к несуществующему серверу

- Превышение времени ожидания ответа
